# The Odin Project: Landing Page

## Project Overview

This repository contains my solution for the landing page project by The Odin Project, mostly used for practice with CSS Flexbox

## Features

This landing page is a static single-page that contains the following:

* **Header:** The header features the logo and some links to direct us to another page/section.
* **Hero:** The hero section contains the title, subtitle, button, and image.
* **Information:** The information contains images with subtext explaining the information within that image.
* **Quotes:** The quotes section contains quote, specifically from Quran, as motivation/inspiration.
* **Call-To-Action:** This section contains some text for user engagement, calling people to sign up.
* **Footer:** The footer contains the copyright.

## Technologies Used

* **HTML5:** For the structure of the webpage.
* **CSS3:** For all the styling including fonts, colors, and layouts.
